Empleados

En esta ventana se pueden consultar, o de tener los privilegios apropiados, agregar o eliminar los empleados contenidos en la base de datos.

Para consultar un empleado, basta con que se coloquen los primeros caracteres de la nomina de un empleado para luego utilizar el botón de buscar. Se llenarán los campos con la primera ocurrencia con los datos de dicho empleado.

En cuanto a borrar, basta sea consultado el empleado para así mediante un clic sobre el botón de eliminar, retirar de la base de datos dicho empleado.

 

Coloque el cursor sobre algún campo editable para obtener la descripción del campo a utilizar.

 

Para asignarle a un empleado un usuario y contraseña para darle acceso y privilegios en el sistema, ingrese con un usuario con privilegios de edición de usuarios, de preferencia un administrador.
