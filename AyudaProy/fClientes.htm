Clientes

En esta ventana se pueden consultar, o de tener los privilegios apropiados, agregar o eliminar los clientes contenidos en la base de datos.

Para consultar un cliente, basta con que se coloquen los primeros caracteres del RFC de un cliente para luego utilizar el botón de buscar. Se llenarán los campos con los datos de dicho cliente.

En cuanto a borrar, basta sea consultado el cliente para así mediante un clic sobre el botón de eliminar, retirar de la base de datos dicho cliente.

 

Coloque el cursor sobre algún campo editable para obtener la descripción del campo a utilizar.
