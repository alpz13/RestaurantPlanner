Productos

En esta ventana se pueden consultar, o de tener los privilegios apropiados, agregar o eliminar los productos contenidos en la base de datos.

Para consultar un producto, basta con que se coloquen los primeros caracteres del nombre de un producto para luego utilizar el botón de buscar. Se llenarán los campos con los datos de dicho producto.

En cuanto a borrar, basta sea consultado el producto para así mediante un clic sobre el botón de eliminar, retirar de la base de datos dicho producto.

 

Coloque el cursor sobre algún campo editable para obtener la descripción del campo a utilizar.
